Proyecto Académico

Panamá Turismo

Plataforma web enfocada en promover destinos turísticos de Panamá mediante reservas digitales, paquetes turísticos interactivos y experiencias modernas para los usuarios.

Sobre Nuestro Proyecto

Panamá Turismo es una aplicación web desarrollada como Proyecto #1 de Desarrollo de Software VII. El sistema permite visualizar paquetes turísticos, consultar destinos, gestionar reservas y administrar información mediante una interfaz moderna y dinámica.

Durante el desarrollo se aplicaron tecnologías como HTML5, CSS3, JavaScript y LocalStorage para crear una experiencia visual interactiva y responsiva.
